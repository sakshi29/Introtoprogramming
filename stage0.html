WWW-World Wide Web

It is the collection of HTML documents. It contains hyperlinks i.e. links between pages. HTML is the main content in the web but it can also contain plain text, audio, video and images.

How it works

The user sends HTTP request to the server using computer and internet. The server finds the required HTML documents and sends it back to the user's computer where the web browser diplays the document .

What is HTML

HTML stands for Hyper Text Markup Language. HTML is made up of-
Text content- what you see.
Markup-what it looks like.
References to othe document- images, video etc.
Links to other pages.

Tags, Element and Attributes

Tags define how the web browser format and display the content.Most tags has two part the opening tag and closing tag.
Example <html> </html>
Void tag does not have closing tag since they do not have any content

Attributes provide additional information about the tag.
Example <img src=" "> </img> where src is an attribute of img tag

HTML document is made up of elements.

Inline and Block elements

Html has two type of element inline and block. Inline element only takes as much width as necessary whereas block element form an invisible box around the content. This box has some height and width.
Example of Inline element-
<br>
<span>

Example of Inline element-
<div>
<p>
